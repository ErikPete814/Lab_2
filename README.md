# Lab 2

## Description

This is a simple C++ program that checks if a person is eligible to ride a roller coaster based on their age and height.

## How It Works

The program asks the user to enter:

* Age
* Height (in inches)

It then determines whether the user:

* Meets both the age and height requirements.
* Does not meet the age requirement.
* Does not meet the height requirement.
* Does not meet either requirement.

## Requirements

* C++
* Xcode or any C++ compiler
